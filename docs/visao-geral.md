# Visão Geral do Projeto Duda

## Nome do projeto

**Duda**

## Conceito

O projeto Duda é uma base para um dashboard financeiro pessoal voltado à organização de renda, metas e planejamento de longo prazo.

A primeira aplicação documentada no repositório é o planejamento financeiro da Maria Eduarda, com foco no período dos 15 aos 18 anos.

## Objetivo principal

Criar uma experiência simples, visual e educativa para acompanhar:

- renda semanal e mensal;
- gastos pessoais;
- metas financeiras;
- investimento mensal;
- projeção acumulada;
- evolução até os 18 anos.

## Possível evolução do projeto

O projeto pode evoluir para:

1. Protótipo no Figma.
2. Dashboard web em React.
3. Aplicativo mobile responsivo.
4. Sistema com login.
5. Banco de dados para salvar valores.
6. Exportação de relatórios em PDF.
7. Envio de resumo mensal por WhatsApp.

## Público-alvo inicial

- Jovens em fase de educação financeira.
- Pais ou responsáveis que desejam orientar filhos adolescentes.
- Pessoas que precisam organizar metas simples.
- Projetos educacionais de finanças pessoais.

## Proposta de valor

Transformar planejamento financeiro em algo visual, simples e prático, evitando planilhas difíceis e ajudando o usuário a entender para onde o dinheiro vai.

## Funcionalidades previstas

- Dashboard financeiro.
- Cards de resumo.
- Distribuição mensal por categoria.
- Barras de progresso.
- Metas aos 18 anos.
- Gráficos de projeção.
- Edição de valores.
- Salvamento local ou em banco de dados.
- Layout responsivo.
