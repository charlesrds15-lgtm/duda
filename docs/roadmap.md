# Roadmap do Projeto Duda

## Fase 1 – Organização inicial

Status: **em andamento**

Objetivo: estruturar o repositório e documentar a ideia principal.

Tarefas:

- Criar README inicial.
- Criar documentação do planejamento financeiro.
- Criar prompt para Figma.
- Definir requisitos.
- Definir roadmap.

## Fase 2 – Protótipo no Figma

Objetivo: criar a primeira versão visual do dashboard.

Tarefas:

- Criar frame desktop.
- Criar frame mobile.
- Criar componentes reutilizáveis.
- Criar protótipo interativo.
- Validar hierarquia visual.

## Fase 3 – MVP Web

Objetivo: transformar o design em aplicação web.

Tecnologias sugeridas:

- React;
- Vite;
- TypeScript;
- Tailwind CSS;
- Recharts ou biblioteca similar para gráficos.

Tarefas:

- Criar estrutura inicial do projeto.
- Criar componentes de cards.
- Criar componente de distribuição mensal.
- Criar componente de metas.
- Criar gráfico de projeção.
- Implementar dados estáticos.
- Implementar layout responsivo.

## Fase 4 – Edição de valores

Objetivo: permitir que o usuário personalize o planejamento.

Tarefas:

- Criar formulário de edição.
- Permitir alterar renda.
- Permitir alterar categorias.
- Recalcular totais automaticamente.
- Restaurar valores padrão.
- Salvar no navegador com localStorage.

## Fase 5 – Dados persistentes

Objetivo: salvar informações fora do navegador.

Opções futuras:

- Supabase;
- Firebase;
- banco PostgreSQL;
- API própria.

Tarefas:

- Criar autenticação.
- Criar tabela de usuários.
- Criar tabela de planejamentos.
- Criar tabela de categorias.
- Criar tabela de metas.

## Fase 6 – Relatórios e automações

Objetivo: gerar valor prático recorrente.

Tarefas:

- Criar relatório mensal.
- Exportar PDF.
- Enviar resumo por WhatsApp.
- Criar alertas de metas.
- Criar histórico de evolução.

## Fase 7 – Produto educacional

Objetivo: transformar o dashboard em ferramenta de educação financeira.

Possibilidades:

- Material didático sobre finanças pessoais.
- Guia para jovens.
- Templates de planejamento.
- Versão para pais/responsáveis.
- Versão para escolas ou mentorias.
