# Prompt para refazer o dashboard no Figma

Crie um dashboard financeiro pessoal chamado **Planejamento Financeiro – Maria Eduarda**, para organizar a renda de uma jovem dos 15 aos 18 anos.

## Objetivo

Criar uma interface visual moderna, responsiva e profissional para planejamento financeiro pessoal, com foco em:

- renda mensal;
- distribuição de gastos;
- metas financeiras;
- investimentos;
- projeção acumulada de 36 meses;
- educação financeira.

O design deve parecer um produto digital real, no estilo dashboard/SaaS, e não apenas uma arte estática.

## Tela principal

Crie uma tela desktop de aproximadamente **1440px** de largura, com fundo claro, cards brancos, cantos arredondados, sombra leve, boa hierarquia visual e espaçamento confortável.

### Cabeçalho

Adicionar no topo:

- Badge: **Dos 15 aos 18 anos**
- Título: **Planejamento Financeiro – Maria Eduarda**
- Subtítulo: **Freelance – Garçonete · R$ 80/dia · Qua a Dom**
- Botão: **Editar valores**
- Botão: **Restaurar**

## Cards de resumo

Criar 4 cards principais:

1. **Depósito semanal**
   - Valor: **R$ 400**
   - Texto: **Recebido toda semana**

2. **Renda mensal**
   - Valor: **R$ 1.800**
   - Texto: **Aproximadamente 4 semanas**

3. **Saldo atual**
   - Valor: **R$ 0**
   - Texto: **Disponível hoje**

4. **Patrimônio aos 18**
   - Valor: **R$ 22.680**
   - Texto: **Em 36 meses**

## Distribuição mensal

Criar seção chamada **Distribuição mensal** com a descrição:

**Como os R$ 1.800 mensais são divididos**

Categorias:

| Categoria | Valor | Percentual |
|---|---:|---:|
| Gastos Pessoais & Lazer | R$ 731 | 40,6% |
| Saúde e Bem-Estar | R$ 169 | 9,4% |
| Cursos e Qualificação | R$ 169 | 9,4% |
| Carteira de Motorista | R$ 68 | 3,8% |
| Faculdade | R$ 281 | 15,6% |
| Investimentos e Reserva | R$ 158 | 8,8% |
| Metas de Compras | R$ 225 | 12,5% |

Cada categoria deve ter:

- ícone;
- nome;
- valor mensal;
- percentual;
- barra de progresso fina.

Ao lado, criar um gráfico donut chamado **Proporção visual**.

## Projeção acumulada

Criar gráfico de linha chamado:

**Projeção acumulada – 36 meses**

Subtítulo:

**Renda total no período: R$ 64.800**

O gráfico deve mostrar:

- patrimônio total;
- investimentos.

O eixo horizontal deve representar os meses de 1 a 36.

## Objetivos prioritários

Criar lista com checks:

- Desenvolver educação financeira
- Criar disciplina de poupança
- Construir reserva financeira
- Garantir recursos para habilitação
- Garantir recursos para faculdade
- Investir continuamente em saúde
- Investir em qualificação profissional
- Criar independência financeira gradual
