# Duda

Repositório do projeto **Duda**, uma base para criação de um dashboard financeiro pessoal focado em planejamento, metas, organização de renda e educação financeira.

## Objetivo

Criar uma aplicação visual e simples para acompanhar o planejamento financeiro de uma jovem dos 15 aos 18 anos, começando pelo caso da **Maria Eduarda**.

O projeto pode evoluir para:

- protótipo no Figma;
- dashboard web em React;
- aplicativo responsivo;
- ferramenta de educação financeira;
- sistema com relatórios e automações.
